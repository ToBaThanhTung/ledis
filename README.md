# Ledis 
Ledis is a light-weight Redis-cli in web base.
[Demo](https://ledis-holistics.now.sh/)
## Setup Ledis
```bash
git clone https://github.com/ToBaThanhTung/ledis.git 
```
```bash
npm install
```
for development
```bash
npm run dev
```
build
```bash
npm run build
```
